# Схема Google Таблицы

## Лист `Seating`

Первая строка — заголовки. Порядок колонок фиксирован:

| Колонка | Значение | Пример |
|---|---|---|
| A `Lookup key` | Имя для точного поиска; можно оставить как отображаемое имя | `Анастасия Петрова` |
| B `Display name` | Имя, возвращаемое гостю | `Анастасия Петрова` |
| C `Table number` | Номер или название стола | `4` |
| D `Note` | Необязательная короткая подсказка | `Рядом с окном` |

Клиент никогда не загружает лист целиком. Apps Script сравнивает нормализованное точное имя и возвращает только одну найденную запись.

## Лист `RSVP`

Если лист отсутствует, Apps Script создаст его и заголовки автоматически:

`Timestamp`, `Name`, `Attendance`, `Ceremony`, `Photo session`, `Guest count`, `Drinks alcohol`, `Alcohol preferences`, `Alcohol other`, `Dietary restrictions`, `Comment`.

Не публикуйте таблицу и не выдавайте гостям доступ по ссылке. Права на запись получает только развёрнутый Apps Script от имени владельца.
