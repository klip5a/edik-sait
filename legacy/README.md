# Архив старого кода

Эта папка не используется актуальным приложением и сохранена только как история разработки.

- `static-version/` — прежняя версия сайта на отдельных CSS- и JavaScript-файлах.
- `preact-unused/` — компоненты, заменённые текущими секциями Preact.

Актуальный проект находится в корне репозитория и собирается командой `npm run build`.
